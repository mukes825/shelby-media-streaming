# 🎬 Shelby Media Streaming

### 🌐 Decentralized Media Storage & Streaming Platform Powered by Aptos & Shelby Protocol

A modern Web3 media platform that enables users to securely upload, manage, store, and stream digital content using decentralized infrastructure. Built with React, TypeScript, Aptos blockchain integration, and Shelby Protocol storage technology.

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/mukes825/shelby-media-streaming.git
cd shelby-media-streaming
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build Production Version

```bash
npm run build
```
